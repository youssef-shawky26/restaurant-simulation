# Restaurant Queue Management & Simulation System

## Description
This project simulates a restaurant system that manages dine-in, takeaway, and reserved customers using queue data structures.

## Features
- Multiple queues (dine-in, takeaway, reserved)
- Table allocation system
- Random customer arrival simulation
- GUI using Raylib
- Performance statistics (waiting time, service time)

## Technologies
- C++
- Data Structures (Queues)
- OOP
- Raylib (GUI)
- STL (vector)

1. Compile the code
2. Run the executable
